# Emulator
C# Forms emulator that plays the original Space Invaders game.
Check the data folder, as it includes a test file to get you started.


![](Emulator/Emulator.jpg)

Keys :- 
C- coin
1 - One player
2 - Two player
Left arrow - Move left
Right arrow - Move right
Space - Fire
